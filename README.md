# Proj-Man

A modern desktop application for managing projects efficiently.

## Technologies

- <img height="20" align="center" src="https://devicon-website.vercel.app/api/rust/plain.svg?color=%23DBDBDB"></img> **Rust** - High-performance backend and system integration
- <img height="20" align="center" src="https://devicon-website.vercel.app/api/react/original.svg"></img> **React** - Dynamic user interface
- <img height="20" align="center" src="https://devicon-website.vercel.app/api/typescript/original.svg"></img> **TypeScript** - Type-safe frontend development
- **Tauri** - Lightweight cross-platform desktop framework

## Features

- Create and manage multiple projects
- Create todo lists
- Intuitive and responsive UI
- Fast performance with minimal resource usage
- Cross-platform compatibility (Windows, macOS, Linux)

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run tauri dev`
4. Build for production: `npm run tauri build`
